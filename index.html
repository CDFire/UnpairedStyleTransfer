This is the official GitHub Pages site for the research project by Cole Feuer: Unpaired Style Transfer from 3D Renders to Anime. This project explores how unpaired learning techniques using diffusion models and GANs can convert 3D-rendered animation frames into stylistically cohesive 2D anime frames, while maintaining temporal consistency across sequences.
